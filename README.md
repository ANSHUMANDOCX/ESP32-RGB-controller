# ESP32 WiFi RGB Controller
This is a controller for a 5050 12v RGB leds which is controlled via an Web interface using a ESP32.
### Basic working mechanism
these 5050 led strips are common anode so, using mosfets connected to each colour R G B we can controll the bright ness of each light which in turn will help us choose any coulour we want.

## Parts used
- ESP32 Wroom
- IRFZ44N N-channel Mosfets
- 12V Supply
- 5050 Common anode RGB leds
